# 期末專案

## 架站網址
https://hcxing.azurewebsites.net

## 功能

* 新聞公告 (新增、修改、刪除、查詢)

* 登入、登出 (上方 navbar 的 button 跟著改變)

* 會員管理 (修改、刪除、查詢)

* 航班選擇 (Dropdownlist 的內容跟著改變)

* 相簿管理 (FileUpload 功能)

* 照片展示 (套用 jquery)

* 文字編輯 (套用 CKEdiotor, 沒有連接資料庫)

* AJAX 

## 前端

* 首頁有響應式效果 (頁面寬度縮小 => sidebar)

* CSS：套用 Materialize 的樣板

    來源 -- https://materializecss.com/
